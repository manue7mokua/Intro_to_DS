Here is how I would handle the missing values for each column with missing data:

  • Mocodes (object, 151,741 missing): I’ll fill these with "Unknown" since it’s categorical.

  • Vict Sex (object, 144,765 missing) & Vict Descent (object, 144,777 missing): I plan to impute these with the mode.

  • Premis Cd (float64, 16 missing): I will drop these few rows.

  • Premis Desc (object, 588 missing): I will drop these few rows.

  • Weapon Used Cd (float64, 677,885 missing) & Weapon Desc (object, 677,885 missing): I plan to fill them with "No Weapon".

  • Status (object, 1 missing): I will impute this using the mode.

  • Crm Cd 1 (float64, 11 missing): I plan to fill these with the mode.

  • Crm Cd 2, Crm Cd 3, Crm Cd 4 (high missing values): I will impute with a placeholder or drop these columns since they might lack value.

  • Cross Street (object, 850,909 missing): I’ll replace missing entries with "N/A".

Cleaning the dataset¶

In [4]:
# Fill missing values in 'Mocodes' with "Unknown"
df['Mocodes'] = df['Mocodes'].fillna("Unknown")

# For 'Vict Sex' and 'Vict Descent', fill missing values with the mode
mode_vict_sex = df['Vict Sex'].mode()[0]
df['Vict Sex'] = df['Vict Sex'].fillna(mode_vict_sex)

mode_vict_descent = df['Vict Descent'].mode()[0]
df['Vict Descent'] = df['Vict Descent'].fillna(mode_vict_descent)

# Drop rows where 'Premis Cd' is missing (16 rows)
df = df.dropna(subset=['Premis Cd'])

# Drop rows where 'Premis Desc' is missing (588 rows)
df = df.dropna(subset=['Premis Desc'])

# For 'Weapon Used Cd' and 'Weapon Desc', fill missing values with "No Weapon"
# Note: 'Weapon Used Cd' is numeric, so filling with a string may require type conversion later if needed.
df['Weapon Used Cd'] = df['Weapon Used Cd'].fillna("No Weapon")
df['Weapon Desc'] = df['Weapon Desc'].fillna("No Weapon")

# For 'Status', fill missing values using the mode
mode_status = df['Status'].mode()[0]
df['Status'] = df['Status'].fillna(mode_status)

# For 'Crm Cd 1', fill missing values with its mode
mode_crm_cd1 = df['Crm Cd 1'].mode()[0]
df['Crm Cd 1'] = df['Crm Cd 1'].fillna(mode_crm_cd1)

# For 'Crm Cd 2', 'Crm Cd 3', and 'Crm Cd 4', drop these columns due to high missing values
df = df.drop(columns=['Crm Cd 2', 'Crm Cd 3', 'Crm Cd 4'])
df = df[df["Vict Age"] > 0]

# For 'Cross Street', replace missing entries with "N/A"
df['Cross Street'] = df['Cross Street'].fillna("N/A")

# Save a copy of the cleaned dataset
df_cleaned = df.copy()

# Export the cleaned dataset to a CSV file without the index
df_cleaned.to_csv('Crime_Data_Cleaned.csv', index=False)

  • LAT and LON show a perfect negative correlation (~–1.00), which is expected due to geographic alignment in the dataset — as one coordinate increases, the other tends to decrease.
  • Crm Cd, Crm Cd 1, and Part 1-2 are strongly correlated (~0.7–1.0), indicating they are closely related representations of crime type and either values can be safely dropped.
